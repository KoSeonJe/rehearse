# SEO 기본 설정

## Status: Completed

## Why
현재 index.html에 title만 존재하며 meta description, OG 태그, robots.txt, sitemap.xml이 없음.
검색엔진 노출 및 SNS 공유 시 미리보기가 없는 상태.

## Goal
- 홈 페이지 Google/Naver 검색 노출 최적화
- SNS 공유 시 OG 미리보기 정상 표시
- 크롤러 접근 제어 (protected 페이지 noindex)

## Scope
- Domain: `https://rehearse.co.kr`
- Public page: `/` (HomePage)
- Protected pages: `/dashboard`, `/interview/*` — noindex

## Implementation

### A. index.html 강화
- meta description, OG 태그, twitter card, canonical, theme-color 추가

### B. react-helmet-async 설치 + HelmetProvider 설정
- `main.tsx`에 `HelmetProvider` 추가
- 각 페이지에 `<Helmet>` 적용

### C. robots.txt
- `frontend/public/robots.txt`
- `/dashboard`, `/interview/` disallow

### D. sitemap.xml
- `frontend/public/sitemap.xml`
- 공개 페이지(`/`)만 등록
