# 디자인 시스템 토큰

> UI 일관성을 위한 디자인 토큰을 정의합니다.
> Designer 에이전트가 관리하며, Frontend 에이전트가 구현에 참조합니다.

---

## 컬러

### Primary
| 토큰 | 값 | 용도 |
|------|-----|------|
| `--color-primary` | _(미정)_ | 주요 액션 버튼, 링크 |
| `--color-primary-hover` | _(미정)_ | 호버 상태 |
| `--color-primary-light` | _(미정)_ | 배경 하이라이트 |

### Semantic
| 토큰 | 값 | 용도 |
|------|-----|------|
| `--color-success` | _(미정)_ | 성공, 좋은 피드백 |
| `--color-warning` | _(미정)_ | 경고, 개선 필요 |
| `--color-error` | _(미정)_ | 에러, 나쁜 피드백 |
| `--color-info` | _(미정)_ | 정보성 메시지 |

---

## 타이포그래피

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--font-family` | _(미정)_ | 기본 폰트 |
| `--font-size-sm` | _(미정)_ | 작은 텍스트 |
| `--font-size-base` | _(미정)_ | 본문 |
| `--font-size-lg` | _(미정)_ | 소제목 |
| `--font-size-xl` | _(미정)_ | 제목 |

## 간격 (Spacing)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--space-1` | `4px` | 최소 간격 |
| `--space-2` | `8px` | 요소 내부 간격 |
| `--space-3` | `12px` | 소그룹 간격 |
| `--space-4` | `16px` | 섹션 간격 |
| `--space-6` | `24px` | 큰 섹션 간격 |
| `--space-8` | `32px` | 페이지 레벨 간격 |

## 반응형 브레이크포인트

| 토큰 | 값 | 용도 |
|------|-----|------|

| `--breakpoint-lg` | `1024px` | 데스크톱 |
