# Rehearse Project Evolution & Roadmap

## 1. Milestone History
- **v1**: 기초 템플릿 구조
- **v2**: 모노톤 전환 (Linear/Vercel 스타일 따라하기)
- **v3**: 전문가용 인디고 톤 (Authority-driven) - 너무 차가워졌음
- **v4**: Friendly Companion - 로고 복원 및 따뜻한 톤 복구
- **v5**: Toss-like Minimalism - 정갈한 레이아웃과 가독성 극대화
- **v6**: Rehearse Signature - Electric Violet 컬러 도입 및 시각적 여정 구체화
- **v7 (Current)**: Cinematic AI Studio - 실사 영상 몰입감을 위한 HUD 및 배경 설계

## 2. Key Decisions
- **Logo Over Box**: 로고를 박스에 가두지 않고 헤드라인과 연동하여 브랜드 시인성 확보.
- **Micro-miniatures**: 텍스트 가이드 대신 실제 페이지 디자인을 축소한 Mockup으로 사용자 교육.
- **Dark Studio for Interview**: 면접 시 실제 사람 영상(AI 아바타)의 몰입감을 극대화하기 위해 면접 화면만 시네마틱 다크 모드 채택.

## 3. Future Tasks (Next Step)
- [ ] **AI Video API Integration**: HeyGen 또는 D-ID API를 활용해 면접관을 캐릭터에서 실사 비디오로 교체.
- [ ] **Dynamic HUD**: 실제 분석 엔진과 연동하여 HUD의 게이지(Emotional Pulse 등)가 실시간으로 움직이도록 구현.
- [ ] **PDF Export Design**: 리포트 페이지의 아름다운 미학을 유지한 고품질 PDF 출력 레이아웃 설계.
