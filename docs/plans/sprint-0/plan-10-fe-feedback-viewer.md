## 구현 계획

### PR 1: [FE] 피드백 뷰어 페이지

**데이터:**
- GET /api/interviews/{id}/feedback → 전체 피드백

**레이아웃 (질문세트별 섹션):**

각 질문세트:
- **상단**: 원본 질문 텍스트 + 질문세트 점수 (0~100)
- **왼쪽**: 영상 플레이어
  - MP4 Presigned URL (변환 완료 시) / WebM 폴백
  - 커스텀 타임라인 바:
    - 원본/후속1/후속2/후속3 구간 색상 구분
    - 점수별 색상 (초록=80+, 노랑=50~79, 빨강=~49)
    - 구간 클릭 → 해당 시점 점프
- **오른쪽**: 피드백 패널
  - 원본 답변 / 후속 1~3 탭 분리
  - 각 답변 내 타임스탬프 구간별:
    - STT 텍스트 (필러워드 하이라이트)
    - 언어 점수 + 코멘트
    - 비언어 점수 (시선, 자세, 표정) + 코멘트
    - 구간 종합 코멘트
  - 모범답변/가이드 비교 보기 버튼

**양방향 동기화:**
- 영상 재생 → 현재 구간 피드백 하이라이트 + 자동 스크롤
- 피드백 클릭 → 영상 해당 시점 점프

**실패 대응:**
- MP4 재생 실패 → WebM 폴백
- Presigned URL 만료 → 자동 재발급
- 미분석 질문세트 → "분석되지 않았습니다" + 재분석 요청 버튼

- Implement: `frontend-developer`
- Review: `designer` + `code-reviewer`
