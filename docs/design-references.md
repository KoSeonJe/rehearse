# 웹 디자인 레퍼런스 가이드

이 문서는 서비스 디자인 영감, UI/UX 패턴, 컴포넌트 라이브러리 등 디자인 및 개발 과정에서 참고할 수 있는 핵심 리소스들을 정리한 가이드입니다.

## 1. 웹 디자인 영감 / 쇼케이스
최신 트렌드와 독창적인 레이아웃 영감을 얻을 수 있는 사이트입니다.

*   **[Awwwards](https://www.awwwards.com/)**: 전 세계 최고 수준의 디자인 어워드. 전문가 심사로 보장된 퀄리티.
*   **[Godly](https://godly.website/)**: 크리에이티브한 랜딩페이지와 인터랙션 레퍼런스 중심.
*   **[Siteinspire](https://www.siteinspire.com/)**: 엄선된 웹사이트 갤러리. 양보다 질 위주의 큐레이션.
*   **[Muzli](https://muz.li/)**: 최신 디자인 뉴스와 영감을 제공하는 크롬 확장 프로그램.
*   **[Web Design Inspiration](https://www.webdesign-inspiration.com/)**: 10,000개 이상의 사이트를 업종/색상별로 필터링 가능.

## 2. UI/UX 패턴 라이브러리
실제 앱/웹의 사용자 흐름과 패턴을 분석할 때 유용합니다.

*   **[Mobbin](https://mobbin.com/)**: 30만 개 이상의 실제 앱 스크린샷. "온보딩", "결제" 등 패턴별 검색 가능.
*   **[Refero](https://refero.design/)**: 주요 서비스들의 디자인 시스템 및 UI 비교 분석.
*   **[Page Flows](https://pageflows.com/)**: 유저 플로우(User Flow)를 영상으로 확인 가능.
*   **[UI-Patterns](http://ui-patterns.com/)**: UI 패턴별 문제 정의와 해결책, 예시 정리.
*   **[Banani](https://banani.ai/)**: 2025년 런칭된 AI 기반 디자인 트렌드 분석 플랫폼.

## 3. 개발자용 컴포넌트 & 라이브러리
빠른 구현과 일관된 UI 구성을 위한 도구입니다.

*   **[shadcn/ui](https://ui.shadcn.com/)**: React + Tailwind 기반. 복사해서 사용하는 컴포넌트의 대세.
*   **[HyperUI](https://www.hyperui.dev/)**: 순수 HTML + Tailwind 기반 무료 오픈소스 컴포넌트.
*   **[daisyUI](https://daisyui.com/)**: Tailwind CSS용 시맨틱 클래스 기반 UI 컴포넌트.
*   **[Aceternity UI](https://ui.aceternity.com/)**: Framer Motion 기반의 화려한 애니메이션 컴포넌트.
*   **[Preline UI](https://preline.co/)**: 비즈니스 대시보드 및 전문적인 UI 세트.

## 4. 디자인 시스템 & 국내 레퍼런스
일관된 디자인 원칙과 국내 트렌드 파악에 필수적입니다.

*   **[Design Systems Repo](https://designsystemsrepo.com/)**: 글로벌 대기업들의 디자인 시스템 모음.
*   **[Storybook Showcase](https://storybook.js.org/showcase/)**: 실제 운영되는 서비스들의 스토리북 확인.
*   **[Colors & Fonts](https://www.colorsandfonts.com/)**: 프로젝트 초기 컬러 팔레트 및 타이포그래피 조합 참고.
*   **[DBCUT](https://www.dbcut.com/)**: 국내 최대 웹 디자인 레퍼런스 플랫폼.
*   **[GDWEB](https://www.gdweb.co.kr/)**: 국내 웹 디자인 어워드 수상작 모음.

## 5. AI 도구 & 랜딩페이지 특화
AI를 활용한 빠른 프로토타이핑과 전환율 높은 랜딩페이지 참고용입니다.

*   **[v0.app](https://v0.dev/)**: 프롬프트 기반 React/Tailwind UI 생성 도구 (Vercel).
*   **[Figma Community](https://www.figma.com/community)**: 전 세계 디자이너들이 공유하는 무료 UI 키트 및 리소스.
*   **[SaaS Landing Page](https://saaslandingpage.com/)**: SaaS 제품군에 특화된 랜딩페이지 갤러리.
*   **[Lapa Ninja](https://www.lapa.ninja/)**: 5,000개 이상의 랜딩페이지 레퍼런스.
